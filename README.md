# GuessMyLie

![GuessMyLie Logo](https://guessmyliegame.onrender.com/img/logo.png)

A small multiplayer social game where one player writes three statements and the other must guess the truth.

🎮 **Play here:** [GuessMyLie](https://guessmyliegame.onrender.com)

---

## How to Play

1. Create a room.
2. Share the room code with your friend.
3. One player writes three statements and marks which one is true.
4. The other player tries to guess the truth.
5. Reveal the answer and see if they guessed correctly!

---

## Built With

- Node.js
- Express
- Socket.IO
- HTML / CSS / JavaScript

---

## Multiplayer

Players join using room codes and play rounds of **Two Truths and One Lie** together in real time.
